# python-log-analysis-engine
🐍 Python-based security log analysis tool - detects failed login attempts and suspicious IP addresses

## 📊 Overview
Security automation tool that parses server logs, detects failed login attempts, and flags suspicious IP addresses.

## 🎯 Features
- Parses server logs for failed login attempts
- Flags IP addresses with multiple failures
- Generates security alerts
- Exports suspicious IPs for further investigation
